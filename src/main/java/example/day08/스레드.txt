**프로그램:** 프로그래밍 언어들의 명령어 집합

**프로세스**: 현제 실행중인 프로그램

          - 프로그램 1개 당 프로세스 존재

**멀티 태스킹**: 두 가지 이상의 일/**프로그램 동시 처리**

**멀티 프로세스**: 운영체제가 동시에 여러개의 프로세스 생성해서 처리 기술

-멀티 프로세스가 프로그램 단위의 멀티 태스킹

-멀티 프로세스는 서로 독립적이다.  (서로에게 영향이 없다.) ,크롬 브라우저(프로그램/프로세스)

-만약 카카오톡에서 오류 /문제 발생했을 때 크롬 브라우저는 정상이다.

**프로세스 내에서 실행되는 흐름의 단위**

    -싱글 스레드 : 프로세스 내 실행흐름이 1개이다.

    -멀티 스레드 : 프로세스 내 실행흐름이 2개 이상이다.

    -멀티 스레드가 쓰레드 단위의 멀티 태스킹

    -멀티 스레드는 서로 종속적이다. (서로에게 영향이 있다.)

        -멀티 스레드: 카카오톡(메세지전송, 파일전송, 사진전송 등등)

        -만약에 파일 전송시 오류가 발생하면 카카오톡 프로세스가 종료 되므로 메세지 전송도 같이 종료.

**싱글  스레드 구현**

-자바코드를 컴파일해서 실행하기 윈한 내장 싱글스레드 제공한다.  main 함수
